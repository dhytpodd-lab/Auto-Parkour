# Automatic Parkour Mod for Minecraft 1.21.8 (Fabric)

## 📋 Описание
Мод добавляет авто бота для автоматического паркура в воздухе на блоках, которые появляются под игроком. Для ajParkour плагина для Minecraft.

## ✨ Особенности
- Автоматическое сканирование ближайших блоков
- Интеллектуальный выбор оптимального пути
- Авто-спринт и авто-прыжки
- Настраиваемый HUD с отображением статуса
- Горячие клавиши для всех функций
- Система безопасности (избегание лавы, безопасное падение)
- Полная настройка через конфиг-файл (JSON, либо в самом майнкрафте)

## 🎮 Управление
| Клавиша | Действие |
|---------|----------|
| P | Вкл/Выкл автопаркур |
| K | Вкл/Выкл авто-спринт |
| J | Вкл/Выкл авто-прыжок |
| H | Вкл/Выкл HUD |
| O | Открыть настройки |

## 🔧 Установка
1. Установите Fabric Loader для Minecraft 1.21.8
2. Скачайте Fabric API для версии 1.21.8
3. Поместите мод и Fabric API в папку `.minecraft/mods`
4. Для настроек мода скачайте Cloth Config, иначе будет крашить игру
5. Запустите игру с профилем Fabric
6. Конфиг для сервера grayworld.pro в файлах исходника

## ⚙️ Настройка
Конфигурационный файл при создании находится в `.minecraft/config/autoparkour.json`
Конфигурационный файл для deeplearn при создании находится в `.minecraft/config/autoparkour_learning.json`
Основные параметры:
- `maxScanRange` - радиус сканирования блоков (2-12)
- `autoSprintEnabled` - авто-спринт
- `autoJumpEnabled` - авто-прыжки
- `safeFallEnabled` - безопасное падение
- `hudOpacity` - прозрачность HUD (0-255)
- `diagonalmovement` - Авто-Паркур по диагоналям(опционально, но рекомендую отключить на всякий)
- и многое другое в Конфигураторе....
